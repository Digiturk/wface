# Gereksinimler

WFace ile geliştirme yapabilmek için bilgisayarınızda bazı programların kurulu olması gerekiyor. 

## Node.js

Bilgisayarınızda npm ile ilgili bazı işlemleri yapabilmek için Node.js'in kurulu olması gerekir. 
Lütfen <a href="https://nodejs.org" target="_blank">Node.js</a> adresinden uygulamayı indirin ve bilgisayarınıza kurun.

## .Net Core SDK

WFace uygulamasını bilgisayarınızda çalıştırabilmek için .Net Core ver.>2.1'in yüklü olması gerekir.
Lütfen <a href="https://www.microsoft.com/net/learn/get-started/windows" target="_blank">.Net Core</a> adresinden uygulamayı indirin ve bilgisayarınıza kurun.
