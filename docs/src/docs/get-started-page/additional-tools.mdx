# Ek Araçlar

Geliştirme yaparken kullanabileceğiniz opsiyonel bazı araçları kullanabilirsiniz. 

#### 1. Visual Studio Code

Javascript geliştirme için en uygun geliştirme ortamlarından biridir. 
Buna ek olarak **Atom** veya **Sublime** gibi araçlar da kullanabilirsiniz.
[Visual Studio Code](https://code.visualstudio.com/)'u indirmek için tıklayın.

<br/>

Visual Studio Code içerisinde geliştirme sırasında size yardımcı olabilecek bazı eklentiler: 
* [__GitLens__](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens): Git kullanmayı kolaylaştırır.

#### 2. Git

Eğer kodlarınızı Git üzerinde saklamayı düşünüyorsanız git istemcisini kurmanız gerekmektedir. 
[__Git__](https://git-scm.com/downloads)'i indirmek için tıklayın.

#### 3. React Developer Tools

Uygulamanızın React çıktısını inceleyebileceğiniz yararlı bir Google Chrome eklentisidir. 
[__React Developer Tools__]
(https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi)'i indirmek için tıklayın
