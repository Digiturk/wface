# WFace Framework
WFace Framework, React tabanlı bir önyüz uygulama geliştirme platformudur. 
Tek sayfa uygulaması geliştirme maksatlı kullanım için tasarlanmıştır. İçerisinde bir çok bileşen barındırır.

> WFace'de kullanılan teknolojiler genel olarak __NodeJs__, __React__ ve __Typescript__'dir.

## NodeJs
Nodejs tıpkı JVM ve .NET CLR gibi bir runtime platformudur. 
EcmaScript (Javascript) kodlarımızın çalıştırılmasını sağlar. 
Biz NodeJs'in daha çok paket yönetim sistemi olan npm modülünü kullanacağız. 

> Npm ile ilgili detaylı bilgiye ve komutlara [__buradan__](https://docs.npmjs.com/) erişebilirsiniz.

## React 
React, bileşen tabanlı uygulama geliştirmenize yarayan bir görsel kütüphanedir. 
WFace ile uygulama geliştirebilmek için React'i kullanabiliyor olmak gerekir. 
Eğer React'a aşina değilseniz öncelikle [reactjs.org] (https://reactjs.org/docs/getting-started.html)
adresinden React'ı inceleyebilirsiniz.

## Typescript
Typescript, Microsoft tarafından açık kaynak olarak geliştirilmiş ve 
Javascripte bazı özellikler katan yeni bir programlama dilidir. WFace platformunda tüm kodlamalar 
Typescript ile yapılmakta ve derleme zamanında Javascripte çevrilerek tarayıcı tarafından çalıştırılmaktadır. 

> Typescript ile alakalı daha fazla bilgi için [__buradan__](https://www.typescriptlang.org/) erişebilirsiniz.
