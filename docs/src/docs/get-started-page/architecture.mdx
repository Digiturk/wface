# WFace Framework

WFace Framework, React tabanlı bir önyüz uygulama geliştirme platformudur. 
Tek sayfa uygulaması geliştirme maksatlı kullanım için tasarlanmıştır. İçerisinde bir çok bileşen barındırır.

WFace Framework'ün başlıca özellikleri aşağıdaki gibidir:

1. İçerisinde ihtiyaç duyulabilecek bir çok kullanıcı arayüzü bileşenini barındırır. 
2. Giriş ekranı, ana sayfa, navigasyon gibi temel sayfalar hazırdır. 
3. Herhangi bir arka plan uygulaması ile çalışabilir. Genişletibilir tasarlanmıştır. 
4. Kod üretmek için araçlar içerir.

```jsx
<Other/>
```
