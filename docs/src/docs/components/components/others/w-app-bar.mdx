# WAppBar

WAppBar @material-ui AppBar'dan türetilmiştir.
AppBar'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/app-bar/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
