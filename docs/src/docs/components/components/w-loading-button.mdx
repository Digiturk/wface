# WLoadingButton

WLoadingButton, WButton kullanılarak geliştirilmiştir. Loading özelliğine sahiptir. 

### Ek Özellikler

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
| isLoading           | bool                                                          | false   | Loading animasyonu gösterip göstermeme seçeneği |
| status              | enum(WLoadingButtonStatus): error, normal, success            | normal  | Butonun status değerine göre rengi değişir      |
| progressType        | enum(WLoadingButtonProgressType): circular, linear            | linear  | Butonun loading animasyon tipi                  |
