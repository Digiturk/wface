# WCircularProgress

CircularProgress'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/circular-progress/).

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
