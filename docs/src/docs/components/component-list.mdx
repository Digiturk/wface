# Bileşenler

WFace bileşen kütüphanesinde birçok bileşen bulunmaktadır. 

Bir bileşeni kullanmak için öncelikle onu `import` etmeniz gerekir.

```javascript
import { WButton } from '@wface/components' // Tek bileşen import etmek için
import { WButton, WTextField } from '@wface/components' // Birden fazla bileşeni import etmek için
import * as WFace from '@wface/components' // Tüm bileşenleri tek seferde import etmek için
```
