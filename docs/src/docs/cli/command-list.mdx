# WFace CLI

WFace CLI, WFace ile geliştirilecek projelerde kullanılmak üzere geliştirilmiştir.
Geliştiriciye kurulum, kod üretimi, test, yaygınlaştırma gibi konularda yardımcı olarak kullanılabilir. 

`<>` işaretleri ile tanımlanan parametreler zorunludur. `[]` işareti ile tanımlananlar ise opsiyoneldir. 

| Komut       | Açıklama                                    | Desen                                         | Detaylı Bilgi                                             |
|-------------|---------------------------------------------|-----------------------------------------------|-----------------------------------------------------------|
| create      | Proje ve ekran gibi bileşenler üretilebilir.| `wface create <project|screen|class> [dir]`   | [<WIcon>link</WIcon>](/#/Pages/WFaceCLI/CreateCommand)    |
| install     | client-app uygulamasını indirir.            | `wface install`                               | [<WIcon>link</WIcon>](/#/Pages/WFaceCLI/InstallCommand)   |
| link        | Projeyi WFace'e linkler.                    | `wface link`                                  | [<WIcon>link</WIcon>](/#/Pages/WFaceCLI/LinkCommand)      |
| run         | WFace'i çalıştırır.                         | `wface run`                                   | [<WIcon>link</WIcon>](/#/Pages/WFaceCLI/RunCommand)       |
| uninstall   | client-app uygulamasını kaldırır.           | `wface uninstall`                             | [<WIcon>link</WIcon>](/#/Pages/WFaceCLI/UninstallCommand) |
| update      | client-app uygulamasını günceller.          | `wface update`                                | [<WIcon>link</WIcon>](/#/Pages/WFaceCLI/UpdateCommand)    |
| version     | WFace versiyon bilgilerini yazdırır.        | `wface version`                               | [<WIcon>link</WIcon>](/#/Pages/WFaceCLI/VersionCommand)   |
