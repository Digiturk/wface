# Create

Belirlenen şablonlara göre boş proje veya ekran üretmeye yarar. 

`wface create <project|screen> [dir]` 

Örnekler: 
<CommandHighlight command="wface create project" dscr="Mevcut konuma projeyi oluşturur."/>
<CommandHighlight command="wface create project --dir d:\src" dscr="d:\src konumuna projeyi oluşturur."/>
<CommandHighlight command="wface create screen" dscr="Ekran kodu üretir. Proje kök dizininde çalıştırılmalıdır."/>
<CommandHighlight command="wface create class" dscr="Mevcut konuma boş bir class kodu üretir."/>
<CommandHighlight command="wface create class --dir d:\src\myproject" dscr="d:\src\myproject konumuna boş bir class kodu üretir."/>
