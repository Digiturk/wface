# Link

Ürettiğiniz projenin bilgisayarınıza global olarak yüklenen WFace uygulamasına bağlanması işlemini yapar. 

`wface link`

> Projenizi linklemeden `wface run` komutunu çalıştırırsanız ürettiğiniz ekranları çalıştıramazsınız.

&nbsp;

> Projenizi ürettikten sonra sadece bir defa linklemeniz yeterli olacaktır. Eğer wface/cli yeniden yüklenirse veya güncellenirse yeniden linkleme gerekebilir.
