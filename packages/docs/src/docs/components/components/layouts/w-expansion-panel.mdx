# WExpansionPanel

WExpansionPanel @material-ui ExpansionPanel'dan türetilmiştir.
ExpansionPanel'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/expansion-panels/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
| title               | string                                                        |         | Panelin Başlığı                                 |
| actions             | Array of {text, onClick, custom}                              |         | Panelde kullanılacak butonlar                   |


### Örnekler

<WPlayground header="Temel Kullanım">
<WExpansionPanel title="Panel">
  İçerik
</WExpansionPanel>
</WPlayground><br/>

<WPlayground header="Actions Kullanımı">
<WExpansionPanel title="Panel"
  actions={[
    {text: "Gönder", onClick: () => { alert('ok')}},
    {custom: <WButton><WIcon>send</WIcon></WButton>}
  ]}
>
  İçerik
</WExpansionPanel>
</WPlayground><br/>
