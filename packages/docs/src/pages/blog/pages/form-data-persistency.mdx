# Form Datasının Korunması

Ekran geçişlerinde veya tab içerisinde form kullanımlarında tab geçişlerinde form datalarımız kaybolabilir. 
Bunun önüne geçebilmek için form'a girilen verileri ekranımızın state'inde tutmamız gerekir.
Aşağıdaki örnekde form datası bu şekilde yönetilmiştir. 

<CodeView>
  {`
  <WFace.WForm 
    onSubmit={this.onSubmit} 
    // ilk verimizi ve değişen tüm verileri ekranımıza ait state içerisinde tutuyoruz.
    initialValues={this.state.formData} 
    // form üzerinde gerçekleşen değişimleri state içerine atıyoruz.
    onChange={values => this.setState({formData: values})}
    // initialValues değiştiğinde formu reinitialize olması için ayarlıyoruz. 
    enableReinitialize 
  >
    <WFace.WFormField.TextField label="Adı" name="name" />   
    <WFace.WFormField.Submit>Kaydet</WFace.WFormField.Submit>      
  </WFace.WForm>
  `}
</CodeView>
