# Production Build

WFace ile geliştirdiğiniz projelerinizi test veya production ortamlarına alabilmek için kodlarınızı derlemeniz gerekir. 
Derleme işlemini aşağıdaki adımları izleyerek yapabilirsiniz. 

- Projenizin kök dizininde `npm run build` komutunu çalıştırın. 
- Bu komutun bitmesi ile birlikte `dist` klasöründe derlenmiş static web uygulamanızı bulabilirsiniz.   
- `index.html` dosyasını tarayıcınızda deneyebilirsiniz.
