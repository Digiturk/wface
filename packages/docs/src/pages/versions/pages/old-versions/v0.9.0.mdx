## Zorunlu Değişiklikler

Production build alabilme hatalarının giderilebilmesi için projenizde bulunan `tsconfig.json` dosyasında bulunan
`compilerOptions` elementine aşağıdaki satırları eklemeniz gerekmektedir. 

```javascript
"esModuleInterop": true,
"allowSyntheticDefaultImports": true
```
