# WTextField

WTextField @material-ui TextField'dan türetilmiştir.
TextField'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/text-field/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
| defaultValue        | string                                                        |         | TextField default değeri                        |
| leftButtons         | Array of WTextFieldButton                                     | []      | TextField soluna eklenecek butonlar             |
| rightButtons        | Array of WTextFieldButton                                     | []      | TextField soluna eklenecek butonlar             |
