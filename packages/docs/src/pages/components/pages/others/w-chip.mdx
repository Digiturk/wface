# WChip

WChip @material-ui Chip'den türetilmiştir.
Chip'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/chip/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
