# WTypography

WTypography @material-ui Typography'dan türetilmiştir.
Typography'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/typography/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
