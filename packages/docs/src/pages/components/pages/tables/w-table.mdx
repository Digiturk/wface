# WTable

WTable `material-table`'dan türetilmiştir.
`material-table`'in propertylerine erişmek için [tıklayın](https://github.com/mbrn/material-table).

> material-table [demo](https://mbrn.github.com/material-table) sayfasında bir çok farklı
örnek kullanıma erişebilirsiniz. 

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
