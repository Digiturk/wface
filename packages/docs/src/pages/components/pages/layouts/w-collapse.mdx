# WCollapse

WCollapse @material-ui Collapse'dan türetilmiştir.
Collapse'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/collapse/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |


### Örnekler

</WCollapse>`}>
