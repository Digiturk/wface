# WTabContainer

WTabContainer @material-ui Tabs kullanılarak geliştirilmiştir.
Tabs'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/tabs/).

## WTabPage

WTabPage @material-ui Tab kullanılarak geliştirilmiştir.
Tab'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/tab/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |


### Örnekler

<WPlayground header="Temel Kullanım">
<WCard>
  <WTabContainer defaultValue={1} centered>        
    <WTabPage icon={<WIcon>face</WIcon>}>
      Page 1
    </WTabPage>
    <WTabPage label="Tab2">
      Page 2
    </WTabPage>
    <WTabPage label="Tab3">
      Page 3
    </WTabPage>
  </WTabContainer>
</WCard>
