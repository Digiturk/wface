# WCard

WCard @material-ui Card'dan türetilmiştir.
Card'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/card/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
