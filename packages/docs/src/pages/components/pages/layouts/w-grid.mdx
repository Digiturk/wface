# WGrid

WGrid @material-ui Grid'dan türetilmiştir.
Grid'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/grid/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
