# WDialog

WDialog @material-ui Dialog'dan türetilmiştir.
Dialog'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/dialog/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |


### Örnekler

<WPlayground header="Temel Kullanım" code={
`<WButton onClick={() => this.setState({isOpen: true})}>Dialogu Aç</WButton>
<WDialog 
  open={this.state.isOpen}          
>
  <WDialogTitle> 
    <WTypography color="primary">Başlık</WTypography>
  </WDialogTitle>
  <WDialogContent>            
      <WTextField label="dialog içeriği"/>
  </WDialogContent>
  <WDialogActions>
    <WButton onClick={() => this.setState({isOpen: false})} color="primary">KAPAT</WButton>
  </WDialogActions>
</WDialog>`}>
