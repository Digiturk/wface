# WMessageDialog

WMessageDialog @material-ui Dialog kullanılarak geliştirilmiştir. 
Dialog içerisinde form ve daha detaylı dialoglar için kullanılabilir.

### Ek Özellikler

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                                                    |
|:--------------------|:--------------------------------------------------------------|:--------|:----------------------------------------------------------------------------|
| open                | boolean                                                       | false   | Dialogun açık olup olmama durumunu belirleyen değişken                      |
| title               | string                                                        | ''      | Dialogun bağlığı                                                            |
| text                | string                                                        | ''      | Dialogun içeriği                                                            |
| icon                | 'error' | 'warning' | 'info' | 'success'                      | ''      | Dialogda kullanılacak icon                                                  |
| buttons             | 'YesNo' | 'YesNoCancel' | 'Ok'                                | ''      | Dialogun butonları                                                          |
| onButtonClick       | event: (event:any, button: String) => void                    |         | Dialogun butonlarından herhangi biri tıklandığında çalıştırılacak fonksiyon |

### Örnekler

<WPlayground header="Temel Kullanım" code={
`<WButton onClick={() => this.setState({isOpen: true})}>Dialogu Aç</WButton>
<WMessageDialog 
  title="Başlık"
  open={this.state.isOpen}
  buttons="YesNo"
  onButtonClick={(event, button) => { 
    alert(button + " tıklandı");
    this.setState({isOpen: false});
  }}
  text="WMessageDialog içeriği"
/>`}>
