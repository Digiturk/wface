# WAvatar

WAvatar @material-ui Avatar'dan türetilmiştir.
Avatar'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/avatar/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
