# WList

WList @material-ui List'dan türetilmiştir.
List'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/list/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
|                     |                                                               |         |                                                 |
