# WLoadingButton

WLoadingButton, WButton kullanılarak geliştirilmiştir. Loading özelliğine sahiptir. 

### Ek Özellikler

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
| isLoading           | bool                                                          | false   | Loading animasyonu gösterip göstermeme seçeneği |
| status              | string: error, normal, success                                | normal  | Butonun status değerine göre rengi değişir      |
| progressType        | string: circular, linear                                      | linear  | Butonun loading animasyon tipi                  |
