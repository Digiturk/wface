# WButton

WButton @material-ui Button'dan türetilmiştir.
Button'un propertylerine erişmek için [tıklayın](https://material-ui.com/api/button/).

### Ek Özellikler

Eklenen ve değiştirilen özellikler aşağıdaki gibidir:

| Adı                 | Veri Tipi                                                     | Default | Açıklama                                        |
|:--------------------|:--------------------------------------------------------------|:--------|:------------------------------------------------|
| __color__           | enum: primary, secondary                                      |primary  | Butonlar default olarak primary color ile gelir |
