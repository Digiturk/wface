# Bileşenler

WFace bileşen kütüphanesinde birçok bileşen bulunmaktadır. 

Bir bileşeni kullanmak için öncelikle onu `import` etmeniz gerekir.

```javascript
// Tek bileşen import etmek için
import { WButton } from '@wface/components' 

// Birden fazla bileşeni import etmek için
import { WButton, WTextField } from '@wface/components' 

// Tüm bileşenleri tek seferde import etmek için
import * as WFace from '@wface/components' 
```

Import ettiğiniz bir bileşeni aşağıdaki gibi kullanabilirsiniz: 

<WPlayground header="Örnek WFace Kodu" isCodeOpen>
  <WFace.WButton 
    style={{margin:10}}
    onClick={() => alert('Button 1 tıklandı')}>
    Button 1
  </WFace.WButton>
  <WFace.WButton 
    style={{margin:10}}
    onClick={() => alert('Button 2 tıklandı')}>
    Button 2
  </WFace.WButton>
</WPlayground>
