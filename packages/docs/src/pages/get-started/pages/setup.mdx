#### 1. Yarn

<MultiLang lang="en">
Install yarn to run some commands on it instead of npm.
</MultiLang>
<MultiLang lang="tr">
Npm'e alternatif olarak bazı komutları yarn ile çalıştırdımız için yarn kuruyoruz. 
</MultiLang>


```console
npm i -g yarn
```

#### 2. @wface/cli

<MultiLang lang="en">
Install @wface/cli program that we will use on setup and development.
</MultiLang>
<MultiLang lang="tr">
Kurulum ve geliştirme sırasında kullanacağımız birçok komutu barındıran @wface/cli uygulamasını kuruyoruz. 
</MultiLang>

```console
npm i -g @wface/cli
```

<MultiLang lang="en">
> @wface/cli is a command line application that developed with NodeJs.
To access more detail about @wface/cli and commands [__click__](#/cli).
</MultiLang>
<MultiLang lang="tr">
> @wface/cli, Javascript ile geliştirilmiş bir NodeJs uygulamasıdır.
