# Gereksinimler

WFace ile geliştirme yapabilmek için bilgisayarınızda bazı programların kurulu olması gerekiyor. 

## Node.js

Bilgisayarınızda npm ile ilgili bazı işlemleri yapabilmek için Node.js'in kurulu olması gerekir. 
Lütfen [__nodejs.org__](https://nodejs.org) adresinden uygulamayı indirin ve bilgisayarınıza kurun.

> Nodejs kurma amacımız Javascript ile sunucu uygulaması geliştirmek değildir.
